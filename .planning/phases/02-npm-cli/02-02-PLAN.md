## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-02-A | Tampering | fs-extra copy filter — path traversal via template filename containing ".." | mitigate | The filter function in copyTemplates rejects any src path containing '..' segments; template files are repo-controlled and do not contain traversal sequences, but the filter is belt-and-suspenders |
| T-02-02-B | Tampering | sentinel merge Case C — replaces wrong content if sentinel markers are malformed | mitigate | extractSentinelBlock validates both SENTINEL_START and SENTINEL_END are present before any replacement; if either is missing it falls through to Case B (append), never silently corrupting content |
| T-02-02-C | Information Disclosure | Copy of JOURNAL.md example entry | accept | templates/JOURNAL.md contains placeholder text only (confirmed 01-04-SUMMARY.md); no real credentials or PII |
| T-02-02-D | Tampering | listTemplateFiles used for dry-run output — could expose internal paths | accept | Paths are relative to templateDir; they reveal no system paths beyond the project root where the user intentionally ran goodvibes init |
